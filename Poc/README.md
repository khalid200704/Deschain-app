# Deschain - Proof of Concept

Ini adalah Proof of Concept (PoC) untuk Deschain, sebuah platform pengadaan kolektif untuk UMKM yang didukung oleh AI. Aplikasi ini dibangun menggunakan Next.js dan Tailwind CSS, dengan fokus pada antarmuka yang modern, bersih, dan responsif.

## Fitur Utama

- **Landing Page**: Halaman perkenalan dengan proposisi nilai yang jelas.
- **Dashboard Pengguna**: Pusat kendali bagi pengguna untuk mengelola kebutuhan pengadaan mereka.
- **Manajemen Kebutuhan**: Pengguna dapat menambah, melihat, mengedit, dan menghapus daftar kebutuhan barang/jasa mereka.
- **Grouping Berbasis AI (Simulasi)**: Sebuah tombol untuk memproses kebutuhan dan menampilkan hasil simulasi pencocokan grup.
- **Rekomendasi Vendor (UI)**: Tampilan kartu vendor sebagai placeholder untuk fitur rekomendasi.
- **Chatbot Widget**: Komponen chatbot melayang untuk bantuan instan (hanya UI).
- **Desain Modern**: Menggunakan prinsip UI/UX modern seperti glassmorphism, clean layout, dan microinteractions.

## Cara Menjalankan Proyek

1.  **Install Dependencies**:
    Buka terminal di direktori proyek dan jalankan perintah berikut:
    ```bash
    npm install
    ```

2.  **Jalankan Development Server**:
    Setelah instalasi selesai, jalankan perintah berikut untuk memulai server pengembangan:
    ```bash
    npm run dev
    ```

3.  **Buka Aplikasi**:
    Buka browser Anda dan kunjungi [http://localhost:3000](http://localhost:3000) untuk melihat aplikasi.
